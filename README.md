## Built With

- **Next.js 14** — App Router, TypeScript
- **Tailwind CSS** — styling and layout
- **Framer Motion / CSS animations** — ocean wave and creature animations
- **MongoDB Atlas** — comments and Q&A storage
- **Vercel** — deployment
